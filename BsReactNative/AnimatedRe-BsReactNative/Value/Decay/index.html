Module Value.Decay

Starts with an initial velocity and gradually slows to a stop.

Example
let animatedValue = Animated.Value.create(0.0);

let animation = Animated.decay(
  ~value=animatedValue,
  ~toValue=`raw(1.0),
  ~velocity=100.0,
  (),
);

Animated.start(animation, ());
