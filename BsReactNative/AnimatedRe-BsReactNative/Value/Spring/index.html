Module Value.Spring

Provides a simple spring physics model.

Example
let animatedValue = Animated.Value.create(0.0);

let animation = Animated.spring(
  ~value=animatedValue,
  ~toValue=`raw(1.0),
  ~bounciness=5.0,
  (),
);

Animated.start(animation, ());
