A wrapper for making views respond properly to touches. On press down, the opacity of the wrapped view is decreased, which allows the underlay color to show through, darkening or tinting the view. You can read more on TouchableHighlight component usage in official docs: https://facebook.github.io/react-native/docs/touchablehighlight
